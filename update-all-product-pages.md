# Template pentru actualizarea paginilor de produs

Toate paginile de produs trebuie să folosească același design ca `profile-gips-carton.astro`:

1. CategoryHero în loc de ProductHero
2. ProductSidebar (400px) în loc de sidebar vechi
3. ProductTabs pentru Specification, Drawing, Data Sheet
4. Layout 1320px (sidebar 400px + content 896px)
5. Font Arial pentru titlu
6. QuoteModal pentru cerere ofertă
